ESLCurrentProcess
A class that allows basic control of the current process (this application's process).

CurrentMemoryUsage CurrentPagedMemoryUsage CurrentVirtualMemoryUsage
Is64bitProcess KillProcess ProcessId
ProcessName Restart RestartAsAdministrator

CurrentMemoryUsage
How much, in megabytes, physical memory the current process is using.

CurrentPagedMemoryUsage
The amount of paged memory, in megabytes, that the current process is using.

CurrentVirtualMemoryUsage
The amount of virtual memory, in megabytes, that the current process is using.

Is64bitProcess()
Determines whether the current process is a 32- or 64-bit process.
returns True if the process is 64-bit, False if it isn't.

KillProcess()
Ends the current application's process.

ProcessId
The ID of the current process.

ProcessName
The name of the current process.

Restart(arguments)
Restarts the current process.
arguments Arguments, if any, to pass to the process.

RestartAsAdministrator(arguments)
Restarts the program, requesting administrator permissions when doing so.
arguments Arguments to pass to the process upon start.

returns A reversed string. For example, abcd -> dcba

ToLower(str)
Converts a string to lowercase.
str The string to convert.
returns A lowercase string. For example, AbCD -> abcd

ToUpper(str)
Converts a string to uppercase.
str The string to convert.
returns An uppercase string. For example, abCd -> ABCD
