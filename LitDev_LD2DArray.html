LD2DArray
This object provides a faster way of storing values in a 2D Array.
Internally it uses lists so that the dimension can increase, but it is most efficient to set the size that you will need at the start.
The indexing is by integer starting from 1.

Add Get Set

Add(dim1,dim2)
Create a new 2D array.
It is efficient to initially size the array to a maximum size that will be used, but it will grow as required.
dim1 The initial first dimension size.
dim2 The initial second dimension size.
returns The 2D array name.

Get(arrayName,index1,index2)
Set a value in the 2D array.
arrayName The 2D array name.
index1 The first dimension integer index.
index2 The second dimension integer index.
returns The array value or "" on failure.

Set(arrayName,index1,index2,value)
Set a value in the 2D array.
arrayName The 2D array name.
index1 The first dimension integer index.
index2 The second dimension integer index.
value The value to set.
